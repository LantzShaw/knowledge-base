## 测试方法

每项核查只需 **3 步**：
1. 在手机上点对应路径
2. 看实际行为是否与"预期"列一致
3. 把状态从 `📝` 改成 `✅` / `❌`；失败项截图给我

**优先级建议**（如果不想全测）：
1. 🔴 **P0** — 数据/保存路径（笔记 CRUD / 任务 CRUD / WebDAV 推拉）
2. 🟡 **P1** — AI 流式 + 模型切换（涉及外部网络）
3. 🟢 **P2** — UI 交互（Tab 配置 / 主题 / 调色盘）
4. ⚪ **P3** — 边角（标签云字号 / 计时器跳秒）
